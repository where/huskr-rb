Create a new Status

POST http://huskr.herokuapp.com/api/v1/statuses.json

Post Body:

{
    "status": {
        "title": "zomg title here",
        "user_name": "my_user_name"
    }
}

Expected Response Format:

{
    "created_at": "2012-10-06T21:51:38Z",
    "id": 1,
    "title": "Shhhh... I'm wearing kitten mittens now.",
    "updated_at": "2012-10-06T21:51:38Z",
    "user_name": "kenmazaika"
}
